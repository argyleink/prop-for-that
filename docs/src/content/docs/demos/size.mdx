Two dimension sources. `viewport` (global) writes `--live-vw` / `--live-vh` on resize. `size` (element) writes `--live-w`, `--live-h`, and `--live-aspect` for any bound element, via one shared `ResizeObserver`. Drag the box to resize it, or resize the window.

<SizeDemo />

## The wiring

`viewport` is on by default in auto mode; the element opts into `size` with a `data-prop` attribute — no JavaScript past the import:

```html
<script type="module">import 'prop-for-that/auto'</script>

<div data-prop="size">…</div>
```

```css
/* the heading scales with the element's own width */
.box h2 {
  font-size: calc(var(--live-w) * 0.055px);
}
```
